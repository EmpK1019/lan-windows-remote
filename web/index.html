常规与安全

管理本机身份、访问权限、远程体验和软件更新。

本机信息

设备名称其他局域网电脑会看到这个名称。
临时访问码每 30 分钟自动刷新；到期后旧访问码立即失效。
永久访问密码尚未设置。修改前，已记住此密码的设备可一直免输连接。

连接与权限

允许局域网发现关闭后本机不会广播，也不会接收设备发现信息。
仅允许桌面观看拒绝远程鼠标和键盘输入。
允许锁屏与安全桌面控制需要管理员安装的安全桌面服务。
记住临时访问码直到失效临时访问码只保存在当前软件会话中，并按目标端到期时间清除。

远程体验

画面刷新速度网络较慢时选择“流畅优先”可减少带宽占用。
减少界面动效关闭大部分过渡动画。

系统

登录 Windows 后启动在当前用户登录后自动运行 LAN Remote。
启动时最大化下次启动软件时生效。

软件更新

自动检查 GitHub 最新版本启动后检查远端 Release，不会在后台静默安装。
