# LAN Remote 1.0.12

- 直接恢复并复用 1.0.7 的 Web 远控工具栏、自动隐藏标题栏与锁屏解锁按钮，不再重新绘制近似界面。
- 完整删除 `NativeRemoteToolbar`、`NativeRemoteTitlebar`、`NativeGlassAction` 及其 WinForms 绘制与桥接代码。
- 恢复 1.0.1 的 D3D 视频排除区域合成：原生 H.264 继续负责桌面画面，WebView 负责原有交互控件。
- 工具栏折叠、标题栏显隐、画面适应/填充和锁屏按钮状态变化会同步更新 D3D 排除区域。
- 更新打包版 D3D 窗口回归，验证不存在旧原生悬浮窗，并验证 Web 工具栏能在原生视频上方显示。

本版本不修改 H.264 编解码、30/60/120 FPS、鼠标键盘输入或锁屏解锁流程。
