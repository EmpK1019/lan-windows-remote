# LAN Remote 1.0.5

本版本集中修复锁屏画面、远程解锁、高帧率稳定性和高 DPI 输入问题。

## 主要变化

- 原生 H.264 会话支持正常桌面与 Winlogon 安全桌面的连续切源，切换期间保留上一张有效画面，不再主动清空为黑屏。
- 安全桌面辅助进程可直接输出原生 H.264；Desktop Duplication 不可用时在编码器内部使用 GDI 采集兼容路径。
- 改进锁屏状态判断、首次连接自动解锁和控制中手动解锁流程，输入密码前依据远端 UI 状态选择唤醒、直接输入或重试。
- 30/60/120 FPS 档位提高默认码率，并优先降低实际帧率而不是独立压低画质码率。
- 120 FPS 档位改进硬件/软件编码器选择，避免短基准误选持续吞吐不足的编码器。
- 修复 D3D 呈现阶段重复非阻塞提交导致的可显示帧丢弃。
- 修复 200% 等高 DPI 屏幕上低级鼠标钩子坐标翻倍，点击与滚轮不转发的问题。
- 无边框远控窗口支持四边和四角正常缩放，同时保留原生磨砂工具栏及视频完整显示区域。

## 验证

- Python 自动化测试 78 项通过。
- 原生视频协议和输入传输测试通过。
- 30/60/120 FPS H.264 发布门禁通过；90 Hz 测试屏上 120 档实测渲染约 88 FPS。
- 强制 GDI 安全桌面 H.264 切源与恢复测试通过。
- 打包后的 WebView2、D3D 可见性、高 DPI 鼠标钩子和完整输入链路测试通过。
