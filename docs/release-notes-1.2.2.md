# LAN Remote 1.2.2

- 30/60/120 FPS 档的目标码率调整为 56/80/80 Mbps，实际帧率低于所选上限时不再同步削减码率。
- 30/60 FPS 档最高保留 2560×1440 分辨率，并使用 H.264 High Profile、CABAC、桌面远控场景提示和质量优先编码参数。
- 控制端在 30/60 FPS 档发生画面缩放时启用高质量 D3D 缩放和轻量边缘增强，改善文字与细线条发虚。
- 120 FPS 档保持 1920×1080 上限和低开销编码、渲染策略，在提高清晰度的同时保留高刷新率性能余量。
- 不改动工具栏、输入、鼠标、锁屏解锁及其他远控功能。
