# LAN Remote 1.0.11

- 删除 1.0.8 起启用原生 H.264 后出现的旧 `NativeGlassToolbar` 工具栏结构，不再保留兼容分支。
- 为原生 D3D 远控窗口恢复独立标题栏：设备名称与最小化、最大化/还原、关闭按钮回到窗口顶部。
- 新建原生远控工具条，折叠按钮回到最左侧，窗口按钮不再混入远控操作区。
- 非全屏时工具条定位在标题栏下方；全屏时隐藏标题栏但保留可展开工具条。
- 调整工具条透明度与渐变，改善磨砂玻璃质感，并保持标题栏、工具条和解锁按钮不进入远端输入区域。
- 新增原生窗口壳结构与打包版 D3D 可见性回归，验证标题栏和工具条为两个独立原生覆盖层。

原生 H.264、帧率、自适应质量、鼠标键盘输入及锁屏画面恢复链路均未改动。
