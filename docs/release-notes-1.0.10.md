# LAN Remote 1.0.10

- 修复 1.0.8 起出现的原生远控窗口合成回归：隐藏的 D3D 视频子窗口不再继续向交换链提交画面，避免宿主窗口和原生工具栏被递归卷入远程画面。
- 首帧启动改为“先解码、再显示子窗口、最后提交画面”的两阶段握手，恢复 1.0.7 的窗口层级行为，同时避免首帧渲染死锁和长时间黑屏。
- 锁屏安全源切回桌面源时复用同一首帧握手，保证解锁后的原生 H.264 画面持续更新。
- 新增打包版原生宿主回归：隐藏阶段渲染帧必须为 0，显示后必须稳定渲染，工具栏不得进入捕获画面。
- 新增锁屏恢复双重回归：既检查真实渲染帧持续增长，也检查 D3D 控制画面持续可见更新。
