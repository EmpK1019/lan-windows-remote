# LAN Remote 1.2.3

本版本修复原生 H.264 画面启动与锁屏解锁后的自动恢复问题。

- 修复 D3D 视频面在隐藏状态下丢弃所有首帧，导致原生 H.264 永远无法进入呈现状态的问题。
- 原生视频改为两阶段启用：先保留最新解码帧并唤醒 D3D 呈现，确认已经连续呈现后再切换为原生画面。
- 原生视频的停止、启动和重新连接按顺序执行，避免锁屏切源时的异步竞态。
- MJPEG 兼容模式会持续检查锁屏与安全桌面状态；退出锁屏后自动重试 H.264，不再依赖特定错误文字。
- 明确跟踪锁屏和安全桌面状态转换，解锁完成后立即触发原生画面恢复。
- 增加从隐藏 D3D 视频面启动并恢复呈现的打包端到端回归测试。
